# CodeBook
## Tidy Data Set Variables
* #### subject_id
    A number that identifies the subject who took part int he experiment. Values range from 1 to 30.
* #### activity_name
    A string that indicates the name of the activity that the subject did. 
    Values are :
    * LAYING
    * SITTING
    * STANDING
    * WALKING
    * WALKING_DOWNSTAIRS
    * WALKING_UPSTAIRS
